The scope of var, let, and const

The 'var' is a global scope variable. If var declare in JavaScript it can be access anywhere form function and block scope. Modern JavaScript 'var' can not be used.

The 'let' a is block scope variable. let are only accessible within the block or function where they are defined.

The 'const' a is also block scope variable. But constants are variables whose values cannot be reassigned after they are initially assigned.

The use cases of null and undefined

Null is a value that represents the intentional absence of any object value or a reference to a non-existent or undefined object or entity.
On the other hand, undefined means that the variable has not been declared, or has not been given a value.

Undefined: undefined

What do you mean by REST API?

API stands for Application Programming Interface. And REST API stands for Representational State Transfer Application Programming Interface. APIs can refer to any type of interface that allows communication between different systems. REST APIs, on the other hand, are a specific type of API that adhere to the constraints of the REST architecture.
